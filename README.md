# Sabina Barotova — CV / Portfolio

Личный сайт-портфолио Сабины Баротовой, размещённый через **GitHub Pages**.

## Структура

```
.
├── index.html              # точка входа, выбор направления
├── ba/                     # Business Analyst
│   ├── index.html          # лендинг
│   ├── Resume_Business_Analyst_Barotova_1page.docx
│   └── Resume_Business_Analyst_Barotova_2pages.docx
├── sa/                     # System Analyst
│   ├── index.html          # лендинг
│   ├── Resume_System_Analyst_Barotova_1page.docx
│   └── Resume_System_Analyst_Barotova_2pages.docx
├── assets/                 # PNG-скриншоты учебных кейсов
└── .nojekyll               # отключает Jekyll, чтобы Pages раздавал файлы как есть
```

## Как включить GitHub Pages

После пуша:
1. Settings → Pages
2. Source: **Deploy from a branch**
3. Branch: `main`, Folder: `/ (root)`
4. Save · через 1–2 минуты сайт будет доступен по ссылкам выше.
